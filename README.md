# CSML
This project provides the code and results for 'From Frequency to Shape: Causal-Structural Heterogeneous Mutual Learning for Road Defect Detection in Autonomous Driving'
# Requirements
Python 3.7+, Pytorch 1.5.0+, Cuda 10.2+
If anything goes wrong with the environment, please check requirements.txt for details.

# Data Preparation
The NPO++ dataset can be downloaded from [https://github.com/lab-sun/PotCrackSeg].

After downloading, please extract the dataset into the data/NPO++/ directory under the project root.
